# Portfolio Template

This is a portfolio template showcasing a graduation with a Bachelor of Science in Computer Science, including a dual minor in Artificial Intelligence and Cybersecurity. The portfolio is designed to highlight educational achievements, projects, and contact information for potential employers or collaborators.

## Setup Instructions

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd portfolio-template
   ```

3. Install the dependencies:
   ```
   npm install
   ```

4. Run the development server:
   ```
   npm run dev
   ```

5. Open your browser and go to `http://localhost:3000` to view the portfolio.

## Features

- **Responsive Design**: The portfolio is designed to be responsive and accessible on various devices.
- **Dynamic Components**: Utilize React components for a modular and maintainable codebase.
- **Showcase Projects**: Highlight your projects with detailed descriptions and links.
- **Educational Background**: Clearly outline your educational qualifications and achievements.
- **Contact Information**: Provide a way for potential employers or collaborators to reach out.
